Tentang Saya

Halo! Saya Altharizz, Front-End Developer dari Jember, Jawa Timur. Saya sangat menikmati proses mengubah ide menjadi antarmuka yang indah, cepat, dan mudah digunakan.

Fokus saya saat ini adalah membangun aplikasi web modern menggunakan React ecosystem, Tailwind untuk styling yang cepat, TypeScript untuk kode lebih aman, dan Supabase sebagai backend ringan tapi powerful.

Saya juga suka bereksperimen dengan animasi sederhana, micro-interactions, performance web, dan accessibility agar website bisa dinikmati semua orang.
